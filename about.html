# whoami

Hi, I'm Sudhara Dharmawardhana. I'm a results-driven information security specialist with over five years' experience spanning offensive and defensive security domains. I specialise in comprehensive penetration testing across applications, networks, cloud environments, DevOps, and mobile platforms. I also develop bespoke tools for sophisticated red team engagements. I am also proficient in conducting source code reviews for Go, Python, TypeScript, and Swift-based applications.

In addition to technical skills, my security expertise includes comprehensive knowledge of information security risk management. This broader perspective helps me identify organisational risks and build security architectures that can effectively resist attacks.

Outside the security domains, I enjoy recreational programming, running, fingerstyle guitar, travelling, hiking, and Chess.
